🧟 Zombie Slayer
A browser-based 2D action platformer built with vanilla JavaScript and HTML5 Canvas — no libraries or frameworks used.
🎮 Play the Game
Play Zombie Slayer
🕹️ How to Play

Arrow Keys — Move left and right
Space — Jump
Z — Attack
S — Open/close the upgrade shop

✨ Features

Two playable characters — Paxton and Rosie, each with unique stats and a 25% crit chance for Rosie
8 combat modes — Samurai, Ninja, Gunner, Turtle, Devil, Angel, and a secret unlockable God Mode
4 zombie types — Walker, Runner, Big, and Spitter with ranged attacks
Boss waves every 10 waves with a dedicated health bar
In-game upgrade shop with coin economy and 6 upgradeable stats
Particle effects, floating damage numbers, shield system, and health regeneration
Fully responsive canvas that scales to any screen size

🛠️ Built With

Vanilla JavaScript
HTML5 Canvas API
CSS3
